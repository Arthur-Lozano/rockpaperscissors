# rockpaperscissors

A simple rock paper scissor game built using socket.io

## Setup

### Dependencies

`npm i` - install dependecies.
  - socket.io
  - socket.io-client
  - express
  - readline

`node hub.js` - start server.

`node client.js` - start client and join game.
